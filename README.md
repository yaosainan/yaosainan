## 📚 本科毕业设计

### 🔬 半导体制造过程故障检测方法研究 (2024.12 – 2025.06)

> **针对半导体数据高维(590 维)、非线性、强耦合特点,提出基于 Transformer 的双阶段故障检测框架**

**🏗️ 架构创新**
- 第一阶段 **Balanced Two-Stage Detector** 融合 T²/SPE 双统计量与时间稳定性判据完成初筛
- 第二阶段引入 **Transformer Refiner** 通过自注意力对报警序列做时序精炼

**🧠 算法创新**
- 提出基于重构误差的特征重要性评估机制(**FSR + DP 双指标**)
- Top-K 特征筛选使模型收敛提速约 **30%**

**📊 实验结果**
- **SECOM 半导体数据集**:T² 误报率 / 漏报率均为 **0%**,SPE 误报率 **0.68%**,检测延迟为 0
- **TEP 化工数据集**:多类故障指标全面优于 PCA / KPCA / 自编码器 / OCSVM 四类基线方法
